﻿# Education Finance Intelligence Platform

## Project Overview

The Education Finance Intelligence Platform is an enterprise BI solution designed to centralize K-12 education finance data using Microsoft Fabric, Azure Synapse architecture patterns, Power BI, star schema modeling, and Row-Level Security.

## Technology Stack

- Microsoft Fabric
- Azure Synapse Analytics
- Azure Data Lake Gen2
- Fabric Warehouse
- Power BI
- SQL
- DAX
- Row-Level Security
- Azure DevOps
- GitHub

## Project Status

Current Phase: Phase 1 - Project Initiation and Governance

## Key Deliverables

- Business Requirements Document
- Project Charter
- Data Dictionary
- Star Schema
- Fabric Warehouse
- Power BI Semantic Model
- RLS Design
- Executive Dashboards
- Testing and Validation
- Executive Presentation
